# AlgoScope

AlgoScope is an interactive algorithm visualization and coding practice platform that combines visual learning with hands-on problem solving. Master data structures and algorithms through beautiful visualizations and comprehensive coding challenges.

Demo Link :- https://algoscope.vercel.app/dashboard

## How It Works

### 🎯 Core Functionality
- **Interactive Algorithm Visualizer**: Step-through visualizations for sorting, searching, and data structure operations
- **Coding Practice Platform**: 50+ LeetCode-style problems with multiple difficulty levels
- **Real-time Code Execution**: Run Python, JavaScript, and C++ code directly in your browser using Pyodide
- **Comprehensive Test Cases**: Automated testing with hidden test cases for robust validation
- **Progress Tracking**: Monitor your learning journey and problem-solving statistics

### 🎨 User Experience
- **Dark Theme**: Modern, eye-friendly dark interface with neon accents
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Intuitive Navigation**: Smart sidebar that adapts based on your current activity
- **Interactive UI**: Smooth animations and hover effects throughout the application

### 🛠️ Technical Features
- **Multi-language Support**: Python, JavaScript, and C++ code execution
- **Monaco Editor**: Professional code editing experience with syntax highlighting
- **Split-pane Interface**: Efficient workspace layout for coding and visualization
- **State Management**: Robust state handling with Zustand
- **Type Safety**: Full TypeScript implementation for reliability

## Tech Stack

### Frontend
- **React 18** - Modern React with hooks and concurrent features
- **TypeScript** - Type-safe development
- **Vite** - Lightning-fast build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework with custom design system
- **React Router** - Client-side routing for SPA navigation

### Code Execution & Visualization
- **Pyodide** - Python runtime in the browser for code execution
- **Monaco Editor** - Microsoft's code editor (same as VS Code)
- **React Split** - Resizable panel layouts
- **Custom Visualization Engine** - Built-in algorithm animation system

### UI/UX
- **Radix UI** - Accessible component primitives
- **Material Symbols** - Consistent iconography
- **Google Fonts** - Professional typography (Public Sans, Inter, JetBrains Mono)
- **CSS Animations** - Smooth transitions and micro-interactions

### Development Tools
- **ESLint** - Code linting and formatting
- **PostCSS** - CSS processing and optimization
- **Autoprefixer** - Automatic CSS vendor prefixing
